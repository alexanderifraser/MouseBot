# Mouse Bot PID Controller (Arduino Nano)

## Version
v1.0.0

## Description
This project implements a PID controller to guide a mouse robot using two sensors.

## Features
- PID control (P, I, D tuning)
- Dual sensor input
- Motor speed adjustment

## Hardware
- Arduino Nano
- 2x motors
- 2x sensors

## How it works
The robot calculates the error between left and right sensors and adjusts motor speeds accordingly using PID control.

## Future Improvements
- Auto-tuning PID
- Obstacle avoidance
